# To-Do List Application

Консольное приложение для управления списком задач с сохранением данных в JSON файл.

## Возможности

- Добавление задач с приоритетом (высокий/средний/низкий)
- Отметка о выполнении задач
- Редактирование и удаление задач
- Просмотр активных и выполненных задач
- Статистика по задачам
- Автоматическое сохранение данных

## Запуск через Docker

### Требования

- Docker установлен на системе
- Docker Compose (обычно идёт вместе с Docker)
